# xposed-douyin
基于Xposed框架写的抖音插件

当前适配版本从v11.2.0

**主要功能列表**

> - 直播间关键词回复
> - 群发粉丝消息
> - 直播间自动点赞
> - 直播间自动刷礼物
> - 直播间礼物回谢
> - 直播间进场欢迎
> - 直播间自动刷单
> - 更多...............
